- 소개 -


2018년 한해, 멋쟁이 사자를 하며 코딩만으로 1년을 보낸 컴퓨터 비전공자 3명이 집필한 웹 프로그래밍 계의 혁신적인 도서.
전공자가 아니어도 충분히 해낼수 있다는 것을 보여주는 이 도서는 경영학과 방민석, 경영학과 송혜민, 그리고 기술경영학과 전준혁이 공동 집필했다.
간단한 HTML 부터 시작해 CSS, Ruby, Java Script 까지 한 권으로 모두 끝낼 수 있도록 구성되어 있으며 단순히 지식 전달이 아닌 실제 웹을 기획하고, 제작하며, 익힐 수 있도록 구성되어있다.
